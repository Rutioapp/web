# Landing Page Patterns

## Hero Rules
- Lead with one clear value proposition.
- Use one main CTA.
- Add a secondary CTA only when it improves clarity.
- Avoid overpromising outcomes.

## Section Rules
- Each section explains one clear idea.
- Keep headings short.
- Keep paragraphs short.
- Use scannable cards.
- Keep max widths consistent.
- Maintain balanced desktop composition.
- Build mobile-first and scale up.

## Placeholder Rules
- Use intentional placeholders when final screenshots are missing.
- Describe the expected future screenshot clearly.
- Do not use random stock images.

## CTA Rules
- Keep CTAs aligned with beta/waitlist direction.
- Avoid aggressive conversion language.

## Trust Rules
- Legal, footer, and contact areas should feel complete.
- Do not use fake testimonials.
- Do not use fake social proof.
