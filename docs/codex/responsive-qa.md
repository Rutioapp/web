# Responsive QA Checklist

## Breakpoints To Check
- 375px
- 430px
- 768px
- 1024px
- 1280px
- 1440px

## What To Inspect
- No horizontal scroll
- Hero balance
- CTA wrapping
- Card heights
- Section spacing
- Form usability
- Footer links
- Placeholder cards

## Desktop Expectations
- Sections should not feel stretched.
- Max widths should feel intentional.
- Card grids should align cleanly.

## Mobile Expectations
- Readable text
- Tappable controls
- No dense paragraphs
- No awkward overflow

## Validation Flow
- Run `npm run dev` for visual checks.
- Run `npm run build` for production safety.
- Run `npm run lint` if available.
