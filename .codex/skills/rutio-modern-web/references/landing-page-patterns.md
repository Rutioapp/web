# Landing Page Patterns

## Hero
- Lead with one clear value proposition.
- Use one primary CTA.
- Add a secondary CTA only if it improves clarity.
- Avoid overpromising outcomes.

## Sections
- One core idea per section.
- Short headings and short paragraphs.
- Scannable card-style content where helpful.
- Consistent max-width rhythm and spacing.
- Mobile-first structure, then scale to desktop.

## Placeholders
- Use intentional placeholders when screenshots are not final.
- Label placeholders with expected future content.
- Avoid random stock images.

## CTAs
- Keep CTA intent aligned with beta/waitlist.
- Use calm, clear conversion language.
- Avoid pushy or manipulative phrasing.

## Trust Areas
- Footer/legal/contact areas should feel complete.
- Do not add fake testimonials or fake social proof.
